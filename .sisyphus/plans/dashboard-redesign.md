# 首页改版：数据看板

## TL;DR

> **将首页从"欢迎横幅 + 考试列表"改造为数据仪表盘**，展示统计概览、最近考试动态、精简待考列表。
> 彻底解决与"我的考试"页内容重复的问题。

---

## 现状问题

```
首页 (dashboard)                       我的考试 (paper)
┌───────────────────────┐             ┌─────────────────────┐
│ 欢迎横幅               │             │ 标题 + 考试卡片      │
│ 我的考试 → 考试卡片 👈 │────一样────│ → 考试卡片          │
└───────────────────────┘             └─────────────────────┘
```

同样的 API（`getAvailable()`）、同样的卡片布局，两个页面功能完全重叠。

## 改动文件

### 1. `src/views/dashboard/index.vue` — 核心改动

**Template**: 全新布局
- 欢迎横幅保留（名人名言替代原来的"祝你学习愉快"）
- 新增 3 个统计卡片行
- 新增"最近考试"动态列表
- 待考列表只显示前 2 条 + "更多考试 →" 链接

**Script**: 新增 API 调用
- `examPaperAnswerApi.pageList({page:1, size:3})` → 最近记录 + 总数
- `questionAnswerApi.pageList({page:1, size:1})` → 错题总数
- 已有 `examPaperAnswerApi.getAvailable()` → 截取前 2 条

**CSS**: 新增统计卡片 + 记录行样式（VisionOS 毛玻璃）

---

## 数据来源

| 数据 | API | 响应格式 |
|------|-----|---------|
| 已考总数 | `examPaperAnswerApi.pageList()` | `re.data.total` |
| 最近3条记录 | `examPaperAnswerApi.pageList({size:3})` | `re.data.records` |
| 错题总数 | `questionAnswerApi.pageList({size:1})` | `re.data.total` |
| 待考列表 | `examPaperAnswerApi.getAvailable()` | `re.data[]` 取前 2 |
